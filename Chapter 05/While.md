# While

While is a structure to iterate from one value to other, it is used to repeat a code for
a given number of times.

Example:
```sh
p := uniform [0.3, 0.7];
i := 0;
while i < 2 do 
    result := 0 [p] 1;
    leak result;
    i := i + 1;
od;
```
