# And

The And operator is used to verify if two conditions are true in the same time.

## Example - Comparison Of Numbers

Program:
```python
x = uniform [1, 2, 3];
y = (x < 3) && (x > 1);
```

Output or variable y:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 0.67 | False |
|  | 0.33 | True |
