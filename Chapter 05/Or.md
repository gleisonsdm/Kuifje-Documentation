# Or

The Or operator is used to verify if at least one of two conditions are valid.

## Example - Comparison Of Numbers

Program:
```python
x <- uniform [1, 2, 3];
y = (x < 2) || (x > 2);
```

Output or variable y:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 0.33 | False |
|  | 0.67 | True |
