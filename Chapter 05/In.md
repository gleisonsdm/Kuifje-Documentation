# In

Verifies if a value is in a given subset.

## Example - Comparison Of Numbers

Program:
```python
x = set {1, 2, 3};
y = 2 in x;
z = 4 in x;
```

Output or variable y:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 1.00 | True |

Output of variable z:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 1.00 | False |
