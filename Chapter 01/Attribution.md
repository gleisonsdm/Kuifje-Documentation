# Attribution

To assign a value in Kuifje, the symbol "=" is used.

```python
  x = uniform [1,2,3];
```

The output of this command for variable "x" is:
| Probability of the disitrbution happen | Probability distribution | Value | 
| --- | --- | --- |
| 0.33 | 1 | 1 |
| 0.33 | 1 | 2 |
| 0.33 | 1 | 3 |
