# Uniform Assignment

Describes an assignment where each value has the same probability

Example:
```sh
  x := ['A', 'B', 'B', 'C']
```

It is equivalent to:
| Value | Probability |
| --- | ----------- |
| A | 1/4 |
| B | 1/2 |
| C | 1/4 |
