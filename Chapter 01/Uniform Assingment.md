# Uniform Assignment

Describes an assignment where each value has the same probability

Example:
```sh
  x := [('A',1), ('B',2), ('B',2), ('C',3)]
```

It is equivalent to:
| Value | Probability |
| --- | ----------- |
| 1 | 1/4 |
| 2 | 1/2 |
| 3 | 1/4 |
