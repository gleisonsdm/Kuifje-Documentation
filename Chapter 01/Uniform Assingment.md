# Uniform Assignment

Describes an assignment where each value has the same probability

Example:
```sh
  x = uniform [1, 2, 2, 3];
```

It is output follows:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 0.25 | 1.0 |
| | 0.50 | 2.0 |
| | 0.25 | 3.0 |
