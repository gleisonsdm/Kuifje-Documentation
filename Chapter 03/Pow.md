# Pow operator

Pow is used to estipulate the number of times a value should be multiplied by itself.

## Example Integer
Program:
```python
x = 2 ^ 3;
```

Output or variable x:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 1.00 | 8.0 |

```python
x = uniform [1, 2, 3];
```
