# Remainder operator

Remainder is used to find the value left over a division.

## Example:
Program:
```python
x = 5 % 3;
```

Output or variable x:
| Probability of the disitrbution happen | Distribution | Value | 
| --- | --- | --- |
| 1.00 | 1.00 | 2.0 |
