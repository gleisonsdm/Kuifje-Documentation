# Statement IF

Ifs can be constructed as Expressions or Statements. Statement Ifs represents a decision where an adversary
knows where the value has came from.

Example:
```sh
y = [1, 1, 2, 3]
if y > 2:
  x = 2 + 1
elif y == 2:
  x = 4
else:
  x = 1
```
